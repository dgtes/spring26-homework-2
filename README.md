# Intro to Git: Homework 2

This repository is meant to simulate an *integration manager* workflow.

## Task

1. Explore the list of [open issues on this kanban board](https://github.com/orgs/intro-to-git/projects/2).
2. Choose an issue you want to complete and assign yourself to it.
3. Fork this project into your personal github space
4. Clone **your fork** of the repository to your computer
5. Create a feature branch for your issue
6. Write the code required to resolve the issue
7. Push your feature branch and open a Pull Request
8. Make sure all check on Pull Request are passing
9. Ask a fellow student to review your code
(Everyone should review at least one Pull Request)
10. Submit a link to your Pull Request on Canvas
